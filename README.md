# seasonality

## Python venv setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```


## Example usage

```bash
python3 app.py
```

This will start a local web server. 
You can access the application by navigating to `http://127.0.0.1:5004/` 
in your web browser.
